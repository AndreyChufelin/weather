## Project setup
```
npm install
```

## Set .env
```
VUE_APP_VUE_WEATHER_API_KEY - https://openweathermap.org/
VUE_APP_VUE_GEO_API_KEY - https://rapidapi.com/wirefreethought/api/geodb-cities/
```
